# 🚀 دليل البدء السريع - Quick Start Guide

## ⚡ **في 5 دقائق فقط!**

---

## 📥 **الخطوة 1: تحميل المشروع**

```bash
# انسخ مجلد ProDownloadManager كاملاً إلى جهازك
```

---

## 💻 **الخطوة 2: فتح في Android Studio**

### **الطريقة:**
1. افتح **Android Studio**
2. `File` → `Open`
3. اختر مجلد `ProDownloadManager`
4. انقر `OK`

### **انتظر...**
سيبدأ Android Studio بـ:
- ✅ تحميل Gradle
- ✅ تنزيل التبعيات (Dependencies)
- ✅ فهرسة المشروع

⏱️ **قد يستغرق 3-5 دقائق في المرة الأولى**

---

## 🔨 **الخطوة 3: بناء APK**

### **طريقة سريعة:**

#### **من القائمة:**
```
Build → Build Bundle(s) / APK(s) → Build APK(s)
```

#### **من Terminal:**
```bash
# Mac/Linux
./gradlew assembleDebug

# Windows
gradlew.bat assembleDebug
```

### **انتظر البناء...**
```
BUILD SUCCESSFUL in 2m 15s
```

---

## 📱 **الخطوة 4: تثبيت التطبيق**

### **الطريقة الأسهل (مع هاتف متصل):**
```
Run → Run 'app'
```
سيثبت التطبيق تلقائياً!

### **أو يدوياً:**
1. ابحث عن الملف:
   ```
   app/build/outputs/apk/debug/app-debug.apk
   ```
2. انسخه إلى هاتفك
3. افتحه وثبّته

---

## ✅ **الخطوة 5: الاستخدام**

### **1. منح الصلاحيات**
عند فتح التطبيق لأول مرة:
- ✅ اقبل صلاحية التخزين
- ✅ اقبل صلاحية الإشعارات

### **2. إضافة تنزيل**
1. اضغط زر **+** الأزرق
2. الصق رابط الملف
3. اضغط **إضافة**

### **3. تمتع!**
- شاهد التقدم الحي
- استخدم الأزرار للتحكم
- تصفح التنزيلات المكتملة

---

## 🎯 **مثال سريع**

### **تنزيل ملف اختبار:**

```
الرابط: https://speed.hetzner.de/100MB.bin
الاسم: test-file.bin
الأجزاء: 8

اضغط "إضافة" وشاهد السرعة! 🚀
```

---

## 🔥 **نصائح للمبتدئين**

### **1. اختيار عدد الأجزاء:**
```
ملف صغير (< 10 MB):     2-4 أجزاء
ملف متوسط (10-100 MB):  4-8 أجزاء
ملف كبير (> 100 MB):    8-16 جزء
```

### **2. WiFi أم Mobile Data؟**
```
WiFi:        سرعة عالية + مجاني
Mobile:      أبطأ + يستهلك رصيد

✓ فعّل "WiFi فقط" للملفات الكبيرة
```

### **3. الأولوية:**
```
🔴 عالية:  لملفات مهمة (تنزّل أولاً)
🟡 عادية:  للاستخدام اليومي
🟢 منخفضة: لملفات غير عاجلة
```

## ❓ **مشاكل شائعة وحلولها**

### **1. "APK لم يتم بناؤه"**
```
✓ تأكد من اكتمال Gradle Sync
✓ جرب: Build → Clean Project
✓ ثم: Build → Rebuild Project
```

### **2. "التطبيق يتوقف عند الفتح"**
```
✓ امنح جميع الصلاحيات
✓ أعد تثبيت التطبيق
✓ تأكد من Android 7+
```

### **3. "التنزيل لا يبدأ"**
```
✓ تحقق من الاتصال بالإنترنت
✓ جرب رابط آخر
✓ تأكد من وجود مساحة كافية
```
